@startuml
actor Pegawai
participant "Halaman Profil / Ubah Password" as ProfilPage
participant Sistem
database Database

Pegawai -> ProfilPage : Masuk ke menu Profil
Pegawai -> ProfilPage : Klik Ubah Password
Pegawai -> ProfilPage : Isi password lama dan baru
Pegawai -> ProfilPage : Klik tombol "Simpan"

ProfilPage -> Sistem : Kirim data ubah password
Sistem -> Database : Ambil data password lama
Database --> Sistem : Password lama (terenkripsi)

Sistem -> Sistem : Verifikasi password lama
alt Jika password lama valid
    Sistem -> Database : Update password dengan yang baru
    Database --> Sistem : Konfirmasi update berhasil
    Sistem -> ProfilPage : Tampilkan pesan "Password berhasil diubah"
    ProfilPage --> Pegawai : Tampilkan notifikasi
else Jika password salah
    Sistem -> ProfilPage : Tampilkan pesan error
    ProfilPage --> Pegawai : Tampilkan pesan "Password lama salah"
end
@enduml
